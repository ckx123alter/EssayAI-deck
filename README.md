# EssayAI — 产品演示 Deck




## 打开方式

双击 `index.html` 即可，在浏览器中全屏演示。

- 点击任意幻灯片缩略图 → 进入演示模式
- 键盘 `←` `→` 或点击屏幕左右侧 → 翻页
- 按 `Esc` → 回到概览页
- 按 `F11`（Windows）/ `Cmd+Ctrl+F`（Mac）→ 浏览器全屏，效果等同 PPT
